# Description

> CodeSlide: A simple code slideshow maker

# Installation

## For Development

1. `git clone git@github.com:AsherJingkongChen/codeslide.git codeslide-folder`
2. `cd codeslide-folder`
3. `npm run dev`
4. Open the browser and go to `http://localhost:8081`

## For build

1. `git clone git@github.com:AsherJingkongChen/codeslide.git codeslide-folder`
2. `cd codeslide-folder`
3. `npm run build`
