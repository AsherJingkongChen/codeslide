# Installation
## For Development
1. `git clone git@github.com:AsherJingkongChen/codeslide-cli.git codeslide-cli`
2. `cd codeslide-cli`
3. Start watching changes: `npm run dev`
4. Run the program: `npm run start`
## For Build
1. `git clone git@github.com:AsherJingkongChen/codeslide-cli.git codeslide-cli`
2. `cd codeslide-cli`
3. `npm run build`
4. All files are in `dist` folder
