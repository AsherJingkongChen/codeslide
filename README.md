# Tech Stack
- Template Engine
- Module bundler
- Rust
- TypeScript (ES6)

# Features
- Read client schema (JSON) and export slide files (HTML)
- Use Webpack as Module Bundler
- Use Askama (Jinja-like) for HTML Templating
- Use SerDe as (De)serializer
- Use file descriptors for I/O only
